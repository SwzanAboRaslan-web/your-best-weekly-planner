## ✨ Features

- 🌍 **Bilingual** — Arabic & English with full RTL/LTR support
- ☁️ **Cloud Sync** via Firebase — same data across all your devices
- 🎨 **12 Animated Backgrounds** — sky, forest, ocean, aurora & more
- 🎵 **6 Ambient Sounds** — rain, waves, piano & more (Web Audio API)
- ⭐ **Habit Tracker** with weekly streaks & progress stats
- 🗓️ **Annual Overview** with monthly progress tracking
- 📤 **Google Calendar Export** (.ics file)
- 🌙 **Dark / Light Mode** with animated sun & moon toggle
- 📱 **Fully Mobile Friendly** — works great on any screen size
- 💾 **Single HTML File** — no installation, no dependencies
