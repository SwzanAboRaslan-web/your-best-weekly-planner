# 🗓️ Your Best Weekly Planner

> The most beautiful weekly planner — everywhere you go, on every device.

## 🌐 Live Demo
👉 [Open the App](https://swzanaboraslan-web.github.io/your-best-weekly-planner/)

---

## ✨ Features

- 🌍 **Bilingual** — Arabic & English with full RTL/LTR support
- ☁️ **Cloud Sync** via Firebase — same data across all your devices
- 🎨 **12 Animated Backgrounds** — sky, forest, ocean, aurora & more
- 🎵 **6 Ambient Sounds** — rain, waves, piano & more (Web Audio API)
- ⭐ **Habit Tracker** with weekly streaks & progress stats
- 🗓️ **Annual Overview** with monthly progress tracking
- 📤 **Google Calendar Export** (.ics file)
- 🌙 **Dark / Light Mode** with animated sun & moon toggle
- 📱 **Fully Mobile Friendly** — works great on any screen size
- 💾 **Single HTML File** — no installation, no dependencies

---

## 🛠️ Built With

| Technology | Purpose |
|-----------|---------|
| HTML / CSS / JavaScript | Structure & Design |
| Firebase Authentication | User login & registration |
| Firebase Firestore | Cloud data storage & sync |
| Web Audio API | Ambient sounds (no external links) |
| SVG Animations | Animated illustrated backgrounds |

---

## 🚀 Getting Started

No installation needed! Just open the link and start planning:

2. Or sign in with Google
