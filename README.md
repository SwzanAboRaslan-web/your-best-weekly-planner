> The most beautiful weekly planner — everywhere you go, on every device.

👉 [Open the App](https://swzanaboraslan-web.github.io/your-best-weekly-planner/)

## ✨ Features

### 📅 Planning
- 📅 **Weekly Schedule** — tap any cell to add an activity with type, color & time
- 🗓️ **Annual Overview** — monthly progress tracking with star ratings
- 📤 **Google Calendar Export** — export your schedule as an `.ics` file

### ⭐ Habits
- ⭐ **Habit Tracker** — create custom habits with target days
- 📊 **Weekly Progress Stats** — score, percentage & streaks per habit
- 💡 **Habit Suggestions** — 28+ popular habits to add with one tap

### 📸 Memories
- 📸 **Memory Journal** — log special moments with date, title & description
- 🖼️ **Photo Attachments** — add a photo to each memory
- 🎨 **Built-in Drawing Canvas** — sketch a drawing with brush size control & eraser

### 🧑 Companion Character (Buddy)
- 🧑 **Optional floating buddy** — a chibi-style animated face that travels with you
- 😊 **Mood-reactive** — happy when you complete habits, sad when you fall behind, celebrating at 100%
- 🎨 **Fully customizable** — choose hair color & hairstyle (3 girl styles, 3 boy styles)
- 💬 **Sends encouragement messages** automatically

### 🔐 Authentication & Accounts
- 📧 **Email & Password** registration and login
- 🔑 **Google Sign-In** support
- 😊 **Custom Nickname** — set a display name instead of showing your email
- 🗑️ **Delete Account** — permanently delete your account and all data with password confirmation
- 🚪 **Guest Mode** — use the app locally without an account

### 🎨 Personalization
- 🌍 **Bilingual** — Arabic & English with full RTL/LTR support
- 🌙 **Dark / Light Mode** with animated sun & moon toggle
- 🎨 **12 Animated Backgrounds** — sky, sunset, sunrise, rain, forest, ocean, aurora, stars & more
- 🎵 **6 Ambient Sounds** — rain, waves, fire, wind, piano & lo-fi (Web Audio API, no external links)
- 🕐 **12h / 24h Time Format** toggle

### ☁️ Sync & Storage
- ☁️ **Cloud Sync** via Firebase — same data across all your devices when logged in
- 💾 **Offline Fallback** — works fully offline with localStorage when Firebase is unavailable
- 🔒 **Privacy First** — data is tied to your account only; email is never displayed in full

### 📱 Technical
- 📱 **Fully Mobile Friendly** — works great on any screen size
- 💾 **Single HTML File** — no installation, no build tools, no dependencies

## 🔧 Self-Hosting

This is a single HTML file — you can host it anywhere:

```
1. Download index.html
2. Upload to GitHub Pages, Netlify, Vercel, or any static host
3. Add your domain to Firebase Console → Authentication → Authorized Domains
4. Done!
```
